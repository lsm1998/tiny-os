## 程序内存划分

1. text 存储机器指令和代码
2. rodata 存储常量以及字符串本身
3. data 存储初始化的数据，全局变量和静态变量
4. bss 存放未初始化或初始化为 0 的全局变量和静态变量

磁盘上的 ELF 文件
+----------------------------------+
| ELF Header                       |
+----------------------------------+
| Program Header Table             |
+----------------------------------+
| .text    代码/指令                |
+----------------------------------+
| .rodata  只读常量/字符串          |
+----------------------------------+
| .data    已初始化全局/静态变量     |
+----------------------------------+
| .bss     只记录大小，通常不存内容  |
+----------------------------------+
| 其他节：symtab / strtab / ...    |
+----------------------------------+
